Add thumbnails of featured image to a column of admin All Posts page. No complecated settings.

== Description ==

Add thumbnails of featured image/eyechatch image to a column of admin dashbord All Posts page. / 管理画面のダッシュボード/投稿一覧のカラムに、アイキャッチ画像のサムネイルを追加表示する。

== Installation ==

To install this plugin, simply upload wp/wp-content/plugins folder by FTP software and activate it from your Wordpress admin page.

インストールするには、wp/wp-content/pluginsのフォルダにFTPソフトなどでアップロードし、Wordpressの管理画面から、有効化してください。
